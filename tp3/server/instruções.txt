Inicialização:

Antes de correr deve-se chamar o comando npm start na diretoria deste documento

Note-se que apenas se corre isto uma vez ao inicio, e se houver erros depois 
na consola sobre modulos


Operação do servidor:

1-entrar na diretoria deste documento

2-correr npm start

3- ir para localhost:3000 no browser preferido

4-se tudo correu bem, deve estar a funcionar
